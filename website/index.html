Build from Source

Advanced
bash 
# Clone repository
git clone https://github.com/r3e-network/neo_cpp.git
cd neo_cpp

# Build with CMake
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)

# Run tests
ctest --parallel

# Install
sudo make install

Requirements

  • GCC 11+ or Clang 13+
  • CMake 3.20+
  • C++20 support
  • vcpkg package manager
