Detailed Description

Service proxy for accessing Neo system services from console.

Overview

The ServiceProxy provides a complete and robust interface for console applications to interact with the underlying Neo system services with proper abstraction.

API Reference

  • Service Access: Get various Neo system components
  • Status: Query system status and information
  • Operations: Perform system operations through proxy

Usage Examples

// Create service proxy
auto proxy = ServiceProxy::Create(neoSystem);
// Get blockchain height
auto height = proxy->GetBlockchainHeight();
// Check if node is running
bool running = proxy->IsNodeRunning();
