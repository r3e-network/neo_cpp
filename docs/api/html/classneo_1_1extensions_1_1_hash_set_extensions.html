Detailed Description

Extensions for hash set operations.

Overview

Provides utility methods for set operations, conversions, and manipulations that complement the standard library set functionality.

API Reference

  • Set Operations: Union, intersection, difference
  • Conversions: To/from vectors, filtering
  • Utilities: Subset checking, element operations

Usage Examples

// Union of two sets
auto result = HashSetExtensions::Union(set1, set2);
// Check if subset
bool isSubset = HashSetExtensions::IsSubsetOf(subset, superset);
// Convert to vector
auto vec = HashSetExtensions::ToVector(hashSet);
