◆ NodeCapabilityType

enum class neo::network::p2p::NodeCapabilityType : uint8_t
strong

Enumeration of node capability types.

These capabilities indicate what services a Neo node provides to the network.

Enumerator
TcpServer 

TCP server capability - node accepts TCP connections.

WsServer 

WebSocket server capability - node accepts WS connections.

Deprecated
This capability is obsolete in Neo N3
DisableCompression 

Disable compression capability - node doesn't support compression.

FullNode 

Full node capability - node has complete current state.

ArchivalNode 

Archival node capability - stores full block history These nodes can be used for P2P synchronization from genesis.

Extension0 

The first extension ID. Any subsequent can be used in an implementation-specific way.

Unknown 

Unknown capability type - for unrecognized capabilities.
