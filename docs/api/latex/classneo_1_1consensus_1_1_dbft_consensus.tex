\doxysubsection{Detailed Description}
d\+BFT (Delegated Byzantine Fault Tolerance) consensus implementation 

This implements the Neo consensus algorithm which provides\+:
\begin{DoxyItemize}
\item Byzantine fault tolerance up to f = (n-\/1)/3 faulty nodes
\item Single block finality
\item Deterministic block production 
\end{DoxyItemize}
