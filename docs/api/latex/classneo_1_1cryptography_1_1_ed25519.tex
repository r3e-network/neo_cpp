\doxysubsection{Detailed Description}
\doxylink{classneo_1_1cryptography_1_1_ed25519}{Ed25519} signature scheme implementation. 

This class provides \doxylink{classneo_1_1cryptography_1_1_ed25519}{Ed25519} digital signature functionality following RFC 8032. \doxylink{classneo_1_1cryptography_1_1_ed25519}{Ed25519} is a high-\/speed, high-\/security signature scheme that is widely used in blockchain and cryptographic applications.
