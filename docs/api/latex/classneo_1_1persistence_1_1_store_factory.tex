\doxysubsection{Detailed Description}
Factory class for creating store providers and stores. 

The \doxylink{classneo_1_1persistence_1_1_store_factory}{Store\+Factory} provides a centralized way to create different types of storage providers (memory, Rocks\+DB, etc.) and manages their lifecycle.
