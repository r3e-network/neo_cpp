\doxysubsection{Detailed Description}
Represents the basic unit that contains all the components required for running of a NEO node. 

The \doxylink{classneo_1_1_neo_system}{Neo\+System} class is the main orchestrator for a Neo node, managing all core components including the blockchain, network layer, memory pool, and plugin system.
