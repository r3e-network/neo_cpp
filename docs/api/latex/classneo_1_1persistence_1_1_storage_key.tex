\doxysubsection{Detailed Description}
Represents a key in the storage for Neo N3. 

In Neo N3, storage keys consist of\+:
\begin{DoxyItemize}
\item Contract ID (4 bytes, little-\/endian)
\item Key data (variable length)
\end{DoxyItemize}

This replaces the old script hash + key format from Neo 2.\+x.
