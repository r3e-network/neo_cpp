\doxysubsection{Detailed Description}
Represents a not found payload. 

This payload is sent in response to Get\+Data messages when the requested inventories are not found. It uses the same structure as \doxylink{classneo_1_1network_1_1p2p_1_1payloads_1_1_inv_payload}{Inv\+Payload}.
