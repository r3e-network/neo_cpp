\doxysubsection{Detailed Description}
Adapter class to provide convenient Put/\+Add methods for \doxylink{classneo_1_1persistence_1_1_data_cache}{Data\+Cache}. 

This adapter wraps \doxylink{classneo_1_1persistence_1_1_data_cache}{Data\+Cache} to provide the Put method that was used in the original implementation but isn\textquotesingle{}t part of the base \doxylink{classneo_1_1persistence_1_1_data_cache}{Data\+Cache} interface.
