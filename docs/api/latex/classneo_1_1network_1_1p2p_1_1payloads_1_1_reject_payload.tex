\doxysubsection{Detailed Description}
Represents a reject payload. 

This payload is used to indicate rejection of an inventory. Note\+: Currently the Neo N3 protocol defines this message command but doesn\textquotesingle{}t specify a payload structure. This implementation provides a base implementation that can be extended if needed.
