\doxysubsection{Detailed Description}
Provides interoperable services for the Neo virtual machine. 

The \doxylink{classneo_1_1smartcontract_1_1_interop_service}{Interop\+Service} class manages all system calls and interoperable services that can be invoked from smart contracts running on the Neo VM.
