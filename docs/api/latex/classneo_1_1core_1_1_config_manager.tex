\doxysubsection{Detailed Description}
Configuration management with environment variable support. 

Supports configuration hierarchy\+:
\begin{DoxyEnumerate}
\item Environment variables (highest priority)
\item Configuration file
\item Default values (lowest priority)
\end{DoxyEnumerate}

Environment variables can be referenced in config as \$\{VAR\+\_\+\+NAME\}
