\doxysubsection{Detailed Description}
Represents a fixed-\/point number of arbitrary precision. 

This class provides a decimal number representation with arbitrary precision using a \doxylink{classneo_1_1_big_decimal_a75eb460cae8c115b19c5b667604ab510}{Big\+Integer} value and a fixed number of decimal places.
